Quick Reference for Supplier Orders

Order Format Example:

Supplier: Nazmi Textile

Product: stone (SKU)

Quantity: 100 pcs

Colors: Assorted

Notes:

  • Always use SKU names when ordering from Nazmi
  • Display names are for website and customer catalog only
  • Minimum order: 100 pieces per color per SKU
  • Contact Nazmi directly for custom color requests
